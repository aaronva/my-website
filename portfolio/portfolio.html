Shelter Spinner
v1.0

This is a game I developed with the intent of exploring Unity and general game development. While it is a 2D game from a game play perspective, I instead used 3D logic to practice working a bit more with full vectors. The game suffers a bit from lack of artistic support; when I pick up this project again, getting some better audio and art assets will be an important goal.
